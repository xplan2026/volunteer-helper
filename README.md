# 志愿填报助手 🎯

重庆市高考志愿填报私人辅助工具。

## 使用场景

- 考生城市：重庆（物理类/理科）
- 分数筛选范围：479-529分
- 意向专业：生物工程、制药工程、铁路、电气自动化、通信工程、人工智能、材料科学与工程

## 数据

- `data/generate_schools.py` — 数据生成脚本

筛选逻辑：
1. 剔除地区：西藏、内蒙、山西、河南、河北、湖南、广西、贵州、上海、安徽
2. 剔除投档比 > 103% 的学校
3. 专业限定7个方向

## 页面

| 页面 | 说明 |
|------|------|
| 📊 总览 | 分数、位次、省控线 |
| 🏫 院校推荐 | 按专业/城市/策略筛选 |
| 📋 志愿方案 | 冲-稳-保三段推荐 |
| ⚖️ 对比工具 | 横向对比 |
| 📋 数据表 | 一本线/二本线数据表浏览 |

## 部署

### Cloudflare Pages + GitHub Pages

1. 推送此仓库到 `github.com/xplan2026/volunteer-helper`
2. Cloudflare Pages 连接 GitHub 仓库
3. 设置自定义域名（可选）

### 访问控制

网站使用前端密码验证，密码设置在 `assets/config.js` 中。
